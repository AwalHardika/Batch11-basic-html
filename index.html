Biografi

Nama saya Awal Rafi Hardika berasal dari Purwokerto. Usia saya sudah menginjak usia 23 tahun. Tempat lahir saya di Banyumas pada tanggal 17 Agustus 2000. Semenjak kecil sampai kelas 3 sd saya tumbuh kembang di purbalingga, kemudian di pertengahan semester kelas 3 saya pindah di banyumas tepatnya di desa Kediri kecamatan Karanglewas. Pengalaman paling menakutkan yang pernah saya alami adalah ketika saya hampir meniggal dunia karena tenggelam di sungai logawa belum lama setelah saya pindah di desa Kediri. Saya diselamatkan oleh seseorang yang kemudian setelah itu keluarga saya mengadakan tasyakuran dan mengaggap bahwa penyelamat saya diangkat menjadi kakak saya. Pendidikan terakhir saya saat ini adalah Sarjana Komputer di Universitas Amikom Purwokerto untuk jurusan informatika pada tahun 2023. Saya memiliki ketertarikan di bidang pemrograman. Hobby saya adalah melakukan kegiatan olahraga seperti jogging santai, besepeda, dan berenang.
